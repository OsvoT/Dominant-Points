Steps to obtain the dominant points of a contour-shapes:
	1.-File.
		1.1.- Open Image to select the binary image (.BMP) to obtain its dominant points. 
			1.1.1.- Search the image or file on the computer.
			1.1.2.- Click "Open".
	2.-Assign the tolerable error T, to share the errors committed by each pair of break points (e.g., T = 25.89 for Africa shape, whose best value is reported by Prasad (2012). Some times we use a T greater than proposed in the literature, this value gives us the results reported in the article and that appear in the last column of the table).
	3.-Click "Run".
		3.1.-BPs Detection.- Show the following information:
				- Image that show how more BPs are added when r decreases.
				- ISE generated by the BPs in the i-th iteration.
				- Number of BPs for a given r.
				- r value of decrease ratio: i.e., if r/2 is equal to .5 then r -> 0, otherwise r -> round(r/2). 
			3.1.1.-Click "BPs Elimination".
		3.2.-BPs Elimination.- Show the following information:
				- Image that show how the number of BPs decrease.
				- ISE generated by removed BPs.
				- Numeber of BPs conserved.
			3.2.1.- Click "BPs Rearrangement".
		3.3.-BPs Rearrangement.- Show the following information:
				- Image on the left show the position of BPs before Rearrangement and that on the right shows the best position of the BPs to obtain the smallest ISE. 
				- ISE on left is the ISE generated by removed BPs and on the right after Rearrangement BPs.
				- In both cases the number of BPs is the same since only the BPs are rearranged.
			3.3.1.- Click "Final Results".
	5.-Clean parameters. Clean all parameters and results in the aplication, just conserve the image. It is used when you want different results (error criteria) using the same image but other value of T.
	6.-Restart aplication. Clean all the information in the application.
	7.-Chain code. Show the chain code associated with the image boundary.


To close the program:
	1.-File.
		2.-Click Exit.
